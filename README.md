A Streamlit dashboard for blockchain transaction risk assessment using graph forensics and AI analysis.

## Features
- Transaction tracing and graphing.
- Forensic rubric scoring (centrality, clustering, etc.).
- Sanctions checks and AI summaries.

## Setup
1. Clone: `git clone https://github.com/yourusername/blockchain-forensics-dashboard.git`
2. Install: `pip install -r requirements.txt`
3. Run: `streamlit run dashboard.py`

See VS Code section above for development optimization.
